MA02PZ

Compute the number of zero rows and zero columns of a complex matrix

[Specification] [Arguments] [Method] [References] [Comments] [Example]

Purpose 

  To compute the number of zero rows and zero columns of a complex
  matrix.
Specification 
      SUBROUTINE MA02PZ( M, N, A, LDA, NZR, NZC )
C     .. Scalar Arguments ..
      INTEGER            LDA, M, N, NZC, NZR
C     .. Array Arguments ..
      COMPLEX*16         A( LDA, * )
Arguments

Input/Output Parameters 

  M       (input) INTEGER
          The number of rows of the matrix A.  M >= 0.

  N       (input) INTEGER
          The number of columns of the matrix A.  N >= 0.

  A       (input) COMPLEX*16 array, dimension (LDA,N)
          The leading M-by-N part of this array must contain the
          matrix A.

  LDA     INTEGER
          The leading dimension of the array A.  LDA >= max(1,M).

  NZR     (output) INTEGER
          The number of zero rows of the matrix A.

  NZC     (output) INTEGER
          The number of zero columns of the matrix A.
